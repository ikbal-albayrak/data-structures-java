Java Veri Yapıları ve Algoritma Çalışması
Bu dosyalar, Bilgisayar Mühendisliği eğitimim süresince geliştirdiğim temel veri yapıları ve algoritmik çözümleri içermektedir. Projeler, bellek yönetimi, 
nesne yönelimli programlama (OOP) ve algoritma verimliliği prensipleri doğrultusunda Java dilinde kodlanmıştır.


İçeriği
1.Bağlı Liste Tabanlı Yığın (Stack)
Dinamik bellek yönetimi kullanılarak, sabit boyut sınırlaması olmadan oluşturulmuş bir LIFO (Last-In-First-Out) yapısıdır.
Klasör: /stack
Temel Fonksiyonlar: * ekle(Eleman): Yığının tepesine yeni veri ekler.
yazdir(): Tüm yığını listeler.

2. Rekürsif Karakter Dizisi Çevirme (String Reverse)
Özyineleme (Recursion) mantığını kavramak amacıyla geliştirilmiş, bir metni döngü kullanmadan tersine çeviren algoritmadır.
Klasör: /Recursive-Functions
Algoritma: Taban durum (base case) ve rekürsif adım kullanılarak dizinin karakterleri sondan başa doğru işlenir.
Karmaşıklık: Zaman Karmaşıklığı

Teknik Yetkinlikler
Dil: Java
Kavramlar: Dinamik Bellek Yönetimi, Rekürsiyon, OOP (Sınıf ve Nesne Yapısı), Veri Kapsülleme.
Araçlar: Eclipse IDE, GitHub Versiyon Kontrol Sistemi.
